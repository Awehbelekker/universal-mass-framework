# MASS Framework Frontend

This will be a React-based UI for managing agents, workflows, and importing projects. The goal is to provide a VS Code-like user experience for:
- Project import/export
- Agent and workflow management
- Real-time status and logs
- Easy onboarding and help

## Planned Features
- File/project explorer (import VS Code projects)
- Agent status dashboard
- Workflow runner and monitor
- In-app documentation and onboarding

## Setup (to be implemented)
- `npm install`
- `npm start`
